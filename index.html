Shoppingo

A holistic interface to compare products, prices, delivery times and more, without the need to open a heap of e-commerce tabs.

Shoppingo pulls live JSON data from a custom created API which takes in name and price of the product as parameters.

Also social login was implemented using Firebase and the application also supports live notifications.

- Find the best deals, even if they are on the same website.
- See prices from all the top websites.
- Compare delivery times to select the fastest one.
- Check out the latest Deals of the Day.
